Redemptions

V4

The Redemption endpoints provide processing loyalty rewards

Endpoints

Verb Endpoint Description
POST /v4/loyalty-accounts/{loyalty-account-id}/redemptions Create a new Loyalty Account points redemption.

Redemptions Properties

Name Description
type Type of vested reward to redeem
  • points
amount Amount of vested reward to redeem

Examples

Create a redemption for the loyalty account 123456
POST  /v4/loyalty-accounts/123456/redemptions

Request

{
  "points": 10.0,
  "type": "Points"
}

Response

HTTP/1.1 200 OK
