Attributes

V1

Attributes allow for easy filtering, sorting, and searching of some of our endpoints. Currently attributes are supported on Movies, Theatres and Showtimes.

Endpoints

Verb Endpoint Description
GET /v1/attributes Get All active Attributes

Attributes Links

Relation Description Verbs Templated
self The current list of attributes results. GET No

Attribute


Attribute Properties

Name Description
id Unique identifier of an attribute
code Unique identifier used as the input value for our include and exclude query strings
name Full name for display
abbreviation Abbreviated name
shortDescription Shortened description
longDescription Full text marketing description
appliesToTheatre Indicates if the attribute can be used with the Theatre Endpoint
appliesToShowtime Indicates if the attribute can be used with the Showtime Endpoint
appliesToMovie Indicates if the attribute can be used with the Movie Endpoint
url AmcTheatres.com marketing page
sort Recommended sort order for attribute display
isPremiumFormat Indicates if the attribute is a premium format
isShowtimeGrouping Indicates if the attribute is a showtime grouping
isAdultSuperivisionPolicy Indicates if the attribute is an adult supervision policy
isDisplayIntended Indicates if the attribute is intended for display
isTitleFilter Indicates if the attribute is a title filter
isShowtimeFilter Indicates if the attribute is a showtime filter
isTheatreFilter Indicates if the attribute is a theatre filter
media.LogoDynamic Logo image
media.HeroDesktopDynamic Hero image for desktop display
media.HeroMobileDynamic Hero image for mobile display
media.BackgroundMobileDynamicUrl Background dynamic image for mobile display
media.BackgroundDesktopDynamicUrl Background dynamic image for desktop display
media.IconDynamic Icon image
media.IconKioskDynamic Kiosk icon image
