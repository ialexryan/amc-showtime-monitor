Concessions Pickup Times

V1

Concessions pickup times available at a theatre.

Endpoints

Verb Endpoint Description
GET /v1/theatres/{theatre-id}/concessions/pickup-times/{year}/{month}/{day} Get all concessions pickup times for a theatre. Use this version if there is no showtime associated with the order, such as an order containing no tickets.
GET /v1/showtimes/{showtime-id}/concession-pickup-times Get all concessions pickup times for a theatre based on a given showtime.

Concessions Pickup Times Properties

Name Description
ConcessionsPickupTimes A list of Concessions Pickup Time Representations returned.

Concessions Pickup Times Links

Relation Description Verbs Templated
self The current list of Theatre Concessions Pickup Times results. GET No

Concessions Pickup Times


Concessions Pickup Times Properties

Name Description
available The number of pickup time slots available.
pickupDateTimeUtc The pickup time in UTC.
pickupDateTimeLocal The theatre's local pickup time.
isDefault Indicator to tell which pickup time should be displayed as the default selection.
