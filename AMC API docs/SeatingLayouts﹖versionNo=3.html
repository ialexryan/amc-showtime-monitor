Seating Layouts

V2 V3

Gets an auditorium's seating layout.

The "accountId" and "loyaltyCardNumber" are optional query string parameters to enable premiere seating availability.
If those are omitted, the user will be treated as a non-member.

Endpoints

Verb Endpoint Description
GET /v3/seating-layouts/{theatre-number}/{performance-number} Get Seating Layout For Performance.

Seating Layouts Properties

Name Description
theatreNumber The requested Theatre Number.
rows The total count of rows.
columns The total count of columns.
seats The list of seats in the layout.
performanceNumber The requested Performance Number.
zones The list of theatre zones.

Seating Layouts Links

Relation Description Verbs Templated
self This seating layout GET No

Seat


Seat Properties

Name Description
row The seat's row
column The seat's column
seatName The displayable seat name
type The type of seat
  • Recliner
  • CanReserve
  • NotASeat (used to persist layout and designate isles)
  • Blocked
  • Wheelchair
  • Companion
  • GeneralAdmission
  • NA
  • LoveSeatLeft
  • LoveSeatRight
seatTier The tier a seat is targeted at.
  • [empty_string_when_non_seat]
  • Regular
  • Premiere

Zone


Zone Properties

Name Description
zoneNumber The zone's number, nullable
surchargeAmount The zone's surcharge. '0' indicates a discount may be applicable.
discountAmount The potentially applicable zone discount.
description The zone's description.
