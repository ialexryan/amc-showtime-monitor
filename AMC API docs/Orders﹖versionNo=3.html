Orders

V3

Endpoints

Verb Endpoint Description
GET /v3/orders/{order-id} Get Order By Order Id.
GET /v3/orders/token-{token} Get Order By Token.
GET /v3/orders/{order-id}/email-{email}/ Lookup Order By Id and Email.
POST /v3/orders Create a new Order.
PUT /v3/orders/{order-id} Update Order Contact Info By Order Id.
PUT /v3/orders/{order-id} Update Order Contact Information By Order Id. Add SMS concessions notification.
DELETE /v3/orders/{order-id} Delete Order By Order Id.
POST /v3/orders/{order-id}/products Add Product to Order.
PUT /v3/orders/{order-id}/products/{line-number} Update Product in Order.
DELETE /v3/orders/{order-id}/products/{line-number} Delete Product from Order.
PATCH /v3/orders/{order-id}/products/{line-number} Update delivery information for concessions and gift card products.
POST /v3/orders/{order-id}/payments Add Payment to Order.
DELETE /v3/orders/{order-id}/payments/{line-number} Delete Payment from Order.
POST /v3/orders/{order-id}/refresh-expiration Refresh Order Expiration Date.
POST /v3/orders/{order-id}/products/{line-number}/sms-ticket-confirmation/{phone-number} Send Ticket Confirmation to a Mobile Phone.
GET /v3/amc-accounts/{account-id}/orders/ Get All Orders For Account Holder.
POST /v3/orders/{order-id}/loyalty-rewards-application Apply available loyalty rewards for the order items.
DELETE /v3/orders/{order-id}/loyalty-rewards-application Remove available loyalty rewards from the order items.
POST /v3/orders/record Record a completed order.
POST /v3/orders/{order-id}/failed-payment-auths Record payment authorization failure.
